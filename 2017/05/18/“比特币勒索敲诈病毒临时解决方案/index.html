比特币勒索敲诈病毒临时解决方案

发表于 2017-05-18 | 分类于 0day事件 | 0 comments

前言

前几天全球有 90 多个国家遭受比特币勒索病毒的攻击，中国的大范围校园网包括公司网络也惨遭攻击。甚至连加油卡服务终端都未能幸免于难。受到攻击的主机磁盘重要文件被加密，并且文件后缀会改为 ONION 或 WNCRY 无法正常打开。必须向黑客支付 5 个比特币和 300 美元才能解密恢复。国内首先出现的是 ONION 病毒，这次攻击毫无疑问是从暗网发起的。病毒是通过前段时间 NSA 泄露的黑客工具 Eternalblue（永恒之蓝）进行传播的，Eternalblue 能攻击 Windwos 的 445 端口，如果用户系统没有安装微软 3 月份发布的 MS17-010 补丁，那么就能被黑客任意执行攻击代码。虽然运营商已经对于用户封闭了 445 端口，但是仍然有很多系统开启 445 端口，尤其是教育网。对于高校快要毕业的学生毕业论文被加密，会直接影响到毕业，所以校园网处于病毒泛滥的重灾区。


漏洞影响范围

此次利用的 SMB 漏洞影响以下未自动更新的操作系统：

  • Windows XP／Windows 2000／Windows 2003
  • Windows Vista／Windows Server 2008／Windows Server 2008 R2
  • Windows 7／Windows 8／Windows 10
  • Windows Server 2012／Windows Server 2012 R2／Windows Server 2016

    个人预防措施

  1. 没有受到攻击的同学最好立即安装微软发布的MS17-010补丁
  2. 关闭445端口 [如何关闭][4]
  3. 不要轻易下载打开不明内容的文件，尤其是doc
  4. 安装360推出的NSA武器库免疫工具，检测系统是否存在被攻击的隐患

    学校预防措施

  5. 在校园网内禁止35/137/139/445端口的连接。
  6. 在边界出口交换路由设备禁止外网对校园网135/137/139/445端口的连接。

    解决方案

  7. 对于遭受病毒的同学不要向黑客支付比特币，即使能支付也不见得能解密文件。
  8. 下面提供一个网上流传的解决方法。
    一:打开自己的那个勒索软件界面，点击copy. （复制黑客的比特币地址）
    二:把copy粘贴到btc.com （区块链查询器）
    三:在区块链查询器中找到黑客收款地址的交易记录，然后随意选择一个txid（交易哈希值）
    四:把txid 复制粘贴给 勒索软件界面按钮connect us.
    五:等黑客看到后 你再点击勒索软件上的check payment.
    六:再点击decrypt 解密文件即可。
    解锁软件：https://github.com/QuantumLiu/antiBTCHack

    结语

    近期的网络安全事件非常多，先是 NSA 公布的黑客武器让一大波 Windows 沦陷，还有 office 和 Wrodpress 的漏洞，也是致命的。紧接着比特币勒索敲诈病毒。希望这次的病毒攻击可以让大家提高安全意识，尤其是教育网的管理员。
