学习层次总结

发表于 2017-05-19 | 分类于 随笔 | 0 comments

首先以我的观念，我心中的几种技术层次，一是菜鸟层、二基础层、三中上层、高手层。

第一层菜鸟层：这类会玩电脑、会玩游戏、会装杀毒、会安装软件、会重装系统、会拆机子、会组装电脑、

第二层基础层：可以熟悉运用各种工具，如sqlmap注入检测工具。awvs burp 菜刀== 网络爬虫工具，拿过一定量的webshell

第三层中上层：熟练各种工具、对漏洞有一定的理解，至少熟练一种脚本语言，如Python==，能自己编写一些简单有用的脚本工具，具有自已的入侵逻辑。有自已独特的思维。穿梭如各大网站之间。

第四层高手层：能否自已挖掘漏洞 并且能够编写出利用的exp，能否看懂别人的漏洞分析。超强的思维能力。各种渗透手法相当娴熟、关于这类 没几年的岁月磨炼是不可能的事情。

注：学无止境！！
